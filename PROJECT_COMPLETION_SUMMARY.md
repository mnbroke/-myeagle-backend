## üéì QUICK START (5-MINUTE SETUP)

### 1. Start Server:
```bash
cd myeagle-backend
node server.js
```

### 2. Create Loyalty Member:
```bash
curl -X POST http://localhost:3001/api/loyalty/member \
  -H "Content-Type: application/json" \
  -d '{"userId":"u1","email":"user@test.com","name":"John"}'
```

### 3. Record Booking:
```bash
curl -X POST http://localhost:3001/api/loyalty/booking \
  -H "Content-Type: application/json" \
  -d '{"userId":"u1","email":"user@test.com","name":"John","bookingAmount":500}'
```

### 4. Generate Referral Link:
```bash
curl -X POST http://localhost:3001/api/referral/generate \
  -H "Content-Type: application/json" \
  -d '{"userId":"u1","userName":"John"}'
```

### 5. View Analytics:
```bash
curl http://localhost:3001/api/analytics/dashboard?period=month
```
